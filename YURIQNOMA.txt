# TozaUy ilovasini real ishga tushirish yo'riqnomasi
# Windows uchun — bosqichma-bosqich

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## 1-QADAM: Dasturlarni o'rnatish (bir marta)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Node.js o'rnatish:
   → https://nodejs.org/en saytiga kiring
   → "LTS" versiyasini yuklab, o'rnating
   → O'rnatib bo'lgach kompyuterni qayta ishga tushiring

2. Git o'rnatish:
   → https://git-scm.com/download/win saytiga kiring
   → Yuklab, standart sozlamalar bilan o'rnating

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## 2-QADAM: Firebase sozlash (bepul)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Firebase — Google'ning bepul real-time database xizmati.
Buyurtmalar shu yerda saqlanadi va real vaqtda ishlovchiga boradi.

1. https://console.firebase.google.com saytiga kiring
   (Google akkauntingiz bilan kiring)

2. "Create a project" tugmasini bosing
   → Loyiha nomi: "tozauy" → Continue
   → Google Analytics: "Enable" → Continue → Create project

3. Firestore Database yarating:
   → Chap menyu: "Firestore Database"
   → "Create database" tugmasi
   → "Start in test mode" tanlang → Next
   → Hudud: "europe-west3" → Enable

4. Web app qo'shing:
   → Chap menyu yuqorisida "</>" (Web) ikonkasini bosing
   → App nickname: "tozauy-web" → "Register app"
   → Ekranda firebaseConfig ko'rinadi — SAQLAB OLING!
   
   Shunday ko'rinadi:
   const firebaseConfig = {
     apiKey: "AIzaSy...",
     authDomain: "tozauy-12345.firebaseapp.com",
     projectId: "tozauy-12345",
     ...
   };

5. src/firebase.js faylini oching va o'sha ma'lumotlarni o'z joyiga qo'ying

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## 3-QADAM: Loyihani ishga tushirish
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. "tozauy" papkasini masalan C:\Projects\ ichiga ko'chiring

2. Windows-da "Command Prompt" yoki "PowerShell" oching
   (Start tugmasi → "cmd" yozing → Enter)

3. Loyiha papkasiga kiring:
   cd C:\Projects\tozauy

4. Kerakli paketlarni o'rnating:
   npm install

5. Ilovani ishga tushiring:
   npm start

6. Brauzer avtomatik ochiladi: http://localhost:3000
   ✅ Ilova ishlayapti!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## 4-QADAM: Internetga chiqarish (Vercel — bepul)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Vercel — kodingizni internetga chiqaradigan bepul xizmat.
Hamma telefonda ochishi mumkin bo'ladi.

1. GitHub akkaunt oching: https://github.com

2. Yangi repository yarating:
   → "+" tugmasi → "New repository"
   → Name: "tozauy" → "Create repository"

3. Loyihani GitHub'ga yuklang (cmd da):
   cd C:\Projects\tozauy
   git init
   git add .
   git commit -m "TozaUy ilova"
   git branch -M main
   git remote add origin https://github.com/SIZNING_NOMINGIZ/tozauy.git
   git push -u origin main

   (SIZNING_NOMINGIZ → GitHub username'ingiz)

4. Vercel'da deploy:
   → https://vercel.com ga kiring
   → "Sign Up" → GitHub bilan kiring
   → "New Project" → GitHub'dan "tozauy" ni tanlang
   → "Deploy" tugmasini bosing

5. 2 daqiqada tayyor! Sizga link beriladi:
   https://tozauy-abc123.vercel.app

   ✅ Shu linkni hamma telefonga yuborishingiz mumkin!
